# Fixed-Global Story Results

This directory holds the curated analysis outputs for the fixed-global versus rotatable comparison.

## Contents
- `presentation_plots/`: main slide figures, including mean axial velocity, velocity variability, depletion peak, swirl, and final nematic order
- `transport_regime/`: control thresholds, run-level and condition-level transport summaries, and medoid kymograph selections
- `kymographs/`: cleaned kymograph panels for `total480`, `mpc12`, `focus40x12`, and the `n=0` control-inclusive subset

These assets are the kept outputs after pruning preview/timecourse-test directories from the repo root.
